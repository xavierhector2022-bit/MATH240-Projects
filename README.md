#  Intro to Computational Mathematics (MATH/CSCI 240)

Welcome to my project repository for **Intro to Computational Mathematics**.  
This course bridges mathematics and programming by applying computational techniques to solve real-world problems across fields like economics, physics, and the life sciences.

---

##  About This Repository

This repository highlights projects focused on implementing mathematical concepts through code. The work emphasizes:

-  Translating mathematical theory into algorithms  
-  Building Python programs to solve quantitative problems  
-  Applying numerical methods to real-world scenarios  
-  Understanding accuracy, efficiency, and limitations of computational approaches  

---

##  Topics Covered

Throughout this course, I explored key areas of numerical analysis and computational methods:

###  Core Numerical Methods
- Taylor Series  
- Root Finding Algorithms  
- Interpolation & Approximation  

###  Calculus-Based Computation
- Numerical Differentiation  
- Numerical Integration  

###  Linear Systems & Optimization
- Solving Linear Equations  
- Minimization Techniques  

###  Differential Equations
- Numerical Solutions to Ordinary Differential Equations (ODEs)
